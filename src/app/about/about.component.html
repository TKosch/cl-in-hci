This project is a living repository of papers relevant to Personal Informatics, a research area which examines technology people use for self-understanding. This repository was originally developed for a literature review mapping contributions to the literature:

Daniel A. Epstein, Clara Caldeira, Mayara Costa Figueiredo, Xi Lu, Lucas M. Silva, Lucretia Williams, Jong Ho Lee, Qingyang Li, Simran Ahuja, Qiuer Chen, Payam Dowlatyari, Craig Hilby, Sazeda Sultana, Elizabeth V. Eikey, Yunan Chen. Mapping and Taking Stock of the Personal Informatics Literature. PACM on Interactive Mobile, Wearable and Ubiquitous Technologies (IMWUT) 4, 4. https://depstein.net/assets/pubs/depstein_imwut20.pdf

If you use this repository in your own work, please cite this paper.

If you wish to add a paper to the repository, you can suggest it by filling out our Google Form. We require all indexed publications be peer reviewed, in English, archival, and relevant to personal informatics. Common publication venues include (but are not limited to) CHI, IMWUT, PervasiveHealth, and CSCW. Any questions about the repository can be sent to Daniel Epstein.

Development of this website was led by Daniel Epstein, with assistance from Simran Ahuja, Payam Dowlatyari, and Sazeda Sultana. The website is hosted on GitHub.
